# Список исправлений модуля Custom Parts

## Хук размера сетки домашней страницы (GridSizeHomePageHook.kt)

| Функция | Статус | Примечания |
|---------|--------|------------|
| Функциональность сетки | Стабильно | Основные изменения размера сетки и корректировки макета работают правильно. |
| Масштаб иконок | Стабильно | Масштабирование иконок стабильно при разных размерах сетки. |
| Режимы отображения текста - Обычный | Стабильно | - |
| Режимы отображения текста - Двойная строка | Стабильно | - |
| Режимы отображения текста - Бегущая строка | Стабильно | Баг: Рабочий стол вылетает из правого края при разблокировке. |
| Изменение сетки | Стабильно | Баг: Вылетает при попытке переместить элементы с колонками меньше 4. |

### Дополнительные заметки

- Нужно добавить переключатель для этого конкретного сегмента: Вкл/Выкл общий кастомный рендеринг.

## Хук масштаба дока (DockScaleHook.kt)

| Функция | Статус | Примечания |
|---------|--------|------------|
| Количество иконок в доке | Стабильно | Пользовательское количество иконок хотсита применяется правильно. |
| Размер иконок в доке | Стабильно | Масштабирование иконок в доке стабильно. |
| Анимации и переходы | Стабильно | Анимации плавающих иконок масштабированы правильно; Переходы перетаскивания работают корректно. |
| Миграция сетки | Стабильно | Логика миграции хотсита обрабатывает пользовательские количества иконок. |
| Интеграция профиля устройства | Стабильно | Настройки InvariantDeviceProfile и DeviceProfile обновляются правильно.
